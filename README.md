# AI Radar

AI Radar — модульный конвейер сбора, нормализации, анализа, валидации, модерации и публикации новостей об искусственном интеллекте.

## Назначение

Проект предназначен для автоматического получения реальных AI-новостей, удаления дублей, подготовки русскоязычных кратких выпусков и их локальной публикации.

## Архитектура

`Collection → Normalize → Analysis → Validation → Moderation → Human Review → Publication`

Дополнительные контуры:

- Pipeline только оркестрирует этапы.
- Scheduler только запускает Pipeline.
- Publication отвечает за публикацию.
- Operations только наблюдает состояние системы.

## Текущий статус
